# ЗаңКеңес AI — Жобаның төлқұжаты (Project Passport)

## 1. Жобаның атауы
**"ЗаңКеңес AI"** — Қазақстан азаматтарына арналған интеллектуалды құқықтық көмек платформасы.

## 2. Жобаның мақсаты
Пайдаланушыларға кез келген уақытта жылдам, сенімді әрі тегін алғашқы құқықтық кеңес беру, күрделі заң терминдерін қарапайым тілмен түсіндіру және күнделікті тұрмыстық немесе бизнеске қатысты заңнамалық сұрақтарды шешуге көмектесу.

## 3. Негізгі аудитория (Target Audience)
- **Қарапайым азаматтар:** Несие, ипотека, алимент, ажырасу, мұрагерлік мәселелерін шешкісі келетін тұлғалар.
- **Шағын және орта бизнес өкілдері (ШОБ):** Салық, кеден, еңбек даулары мен келісімшарттарды реттеу бойынша нұсқаулық іздейтін кәсіпкерлер.
- **Жәбірленушілер:** Тұрмыстық зорлық-зомбылық немесе алаяқтық құрбандары (олар үшін жедел «SOS» батырмасы мен нұсқаулықтар қарастырылған).

## 4. Негізгі құндылық ұсынысы (Value Proposition)
- **Қолжетімділік:** Заңгерге төлейтін қаражаты жоқ адамдар үшін AI арқылы тегін кеңес алу.
- **Тілдік қамту:** Қазақ және орыс тілдеріндегі толыққанды ақпарат.
- **Бір терезе (All-in-One):** Калькуляторлар (салық, алимент), заңгерлер тізімі, кодекстер базасы мен дайын құжат үлгілері бір жерде жинақталған.

## 5. Технологиялық стек
- **Frontend:** React 19, TypeScript, Tailwind CSS
- **State Management:** Zustand
- **Backend / Database:** Firebase (Firestore, Auth)
- **AI Integration:** LLM (Large Language Model) негізіндегі чат
- **Icons & UI:** Lucide React, Framer Motion (анимациялар)

Бұл жоба тек IT өнімі емес, Қазақстан қоғамының құқықтық сауаттылығын арттыруға бағытталған әлеуметтік маңызды платформа болып табылады.
