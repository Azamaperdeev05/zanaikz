# UI/UX Дизайн және Мобильді бейімділік (Design System)

"ЗаңКеңес AI" жобасының визуалды тілі заманауи "Premium WebView" немесе "Apple Human Interface Guidelines" идеологиясына ұқсас етіп жасалған.

## 1. Мобильді-бірінші тәсіл (Mobile-First)
Қазақстанда құқықтық көмек іздейтін адамдардың 80%-дан астамы телефон пайдаланады. Сондықтан қосымша ең алдымен смартфон экранына шақталып кодталған:
- Кең экрандарда (`lg`) контент ортаға келтіріліп, бағаналарға бөлінеді (grid/flex).
- Телефондарда контент бір бағаналы түріне еніп, жан-жақтан тасып кетпеуі (overflow-x-hidden) мұқият қадағаланады.

## 2. Tailwind CSS қолданысы
- **Радиустар (Border Radius):** Барлық карточкалар өте жұмсақ бұрыштармен (rounded-[24px] немесе rounded-[32px]) берілген. Бұл қолданушыға «қауіпсіз, мейірімді қолданба» деген сенім ұялатады (әсіресе заң секілді стрессті тақырыпта).
- **Көлеңкелер (Shadows):** Қарапайым (flat) емес, терең (depth) сезім беру үшін `shadow-md` жұқа border-лармен (`border-black/[0.04]`) араласып қолданылады.
- **Интерактивтілік:** Батырмалар басқан кезде `transition-all` арқылы біртіндеп түсін өзгертеді немесе `scale-[1.02]` арқылы сәл үлкейіп/кішірейеді (Micro-interactions).

## 3. Төменгі навигация (Bottom Navigation Bar)
Мобильді нұсқада қолданушының бас бармағы оңай жетуі үшін негізгі мәзір экранның төменгі жағына орналастырылған.
- **Apple/iOS тәсілі:** Навигация панелі мөлдір (backdrop-blur-xl) етіп жасалған, артындағы контент бұлттыланған (glassmorphism) әсер береді.
- **Padding Bottom (`pb-32`):** Контент (калькулятор немесе чат тарихы) төменгі навигациямен қабаттасып/жабылып қалмауы үшін, барлық басты container-лердің астына жеткілікті бос кеңістік қалдырылған.

## 4. Анимациялар
- `Framer Motion` немесе `animate-in`: Беттер немесе модальді терезелер ашылғанда олар астынан жайлап шығады (slide-in-from-bottom) және көріне бастайды (fade-in). Бұл интерфейстің "өлі" емес, динамикалық екенін білдіреді.

## 5. Түстер палитрасы (Color Palette)
- **Негізгі фон:** Қардай аппақ емес, сәл жылы, көзге жұмсақ `bg-[#f5f5f7]` (сұр-ақ) түсі.
- **Accent Color:** Басты батырмалар мен белсенді иконкалар ашық көк түспен (iOS-тің негізгі акцент түсі).
- **SOS:** Шұғыл көмек функциясы үшін психологиялық әсері күшті `bg-[#ff3b30]` (ашық қызыл) түс. Және оның шрифті барынша таза (white), яғни контраст жоғары.
