# Орнату, Баптау және Болашақ даму жоспары (Deployment & Scaling)

## 1. Жобаны локальды орнату (Local Setup)

Жобаны өз компьютеріңізде іске қосу үшін келесі әрекеттер қажет:
1. **Репозиторийді жүктеу:** `git clone [repository-url]`
2. **Тәуелділіктерді (Dependencies) орнату:** `npm install` немесе `yarn install`
3. **Environment (Орта конфигурациясы):** Түбірлік папкада `.env` файлын құрып, Firebase кілттерін қосу.
   ```
   VITE_FIREBASE_API_KEY="..."
   VITE_FIREBASE_AUTH_DOMAIN="..."
   VITE_FIREBASE_PROJECT_ID="..."
   VITE_FIREBASE_STORAGE_BUCKET="..."
   VITE_FIREBASE_MESSAGING_SENDER_ID="..."
   VITE_FIREBASE_APP_ID="..."
   ```
4. **Іске қосу:** `npm run dev` (Қосымша `http://localhost:3000` (немесе 5173) портында ашылады).

## 2. Өндіріске шығару (Production Deployment)
Vite негізінде жасалғандықтан, өндіріске арналған файлдарды (build) жасау өте жылдам:
- `npm run build` командасын іске қосыңыз. `dist` папкасы пайда болады.
- Бұл `dist` папкасын кез келген статикалық хостингке (Vercel, Netlify, Firebase Hosting) тікелей жүктеп (deploy) жұмыс істете аласыз.

## 3. Болашаққа арналған жоспарлар (Roadmap & Scaling)

1. **Құжат Генераторы (DocGen):**
   Келешекте `Templates` бөлімі тек құрғақ файлды беріп қоймайды. Ол "Бланк толтыру" интерфейсіне айналады. Пайдаланушы атын, ІСН (ИИН) енгізеді, жүйе PDF немесе Word файлына ақпаратты орналастырып (pdf-lib немесе docx кітапханаларымен), дайын дау-арызды генерациялайды.
2. **Кәсіби Заңгерлер Кабинеті (B2B):**
   Заңгерлерге арналған жеке панель (Dashboard) қосу. Заңгерлер сол жерге кіріп, өз қызметтерінің бағаларын өзгертіп, клиенттерден түскен сұранымдарды (leads) қабылдайды.
3. **PWA (Progressive Web App):**
   Қазіргі веб-қосымшаны манифест және сервис уоркерлер (Service Workers) арқылы телефонға (iOS/Android) App Store не Play Market-тенсіз-ақ «Home Screen»-ге бағдарлама ретінде сақтай алатын (Installable) деңгейге көтеру. Оффлайн режимде (Интернетсіз) калькуляторлар мен кодекстерді есептеу мүмкіндігін қосу.
4. **Дауыстық Көмекші (Voice AI):**
   Көру қабілеті нашар немесе көлік айдап бара жатқан азаматтарға заңдық көмекті аудио формада сөйлесіп алуға мүмкіндік беретін "Voice-to-Text" және "Text-to-Voice" моделдерін қосу.
