# API және Сыртқы сервистер (Integrations)

ЗаңКеңес AI платформасы автономды жұмыс істегенімен сыртқы әлеммен бірнеше интерфейстер арқылы байланыса алады. 

## 1. Жасанды Интеллект API-і (OpenAI / Gemini / Genkit)
- **Бағыт:** Жүйенің ішкі чатынан жіберілген (user message) сұрақтарды өңдеу.
- **Логика:** Frontend клиент тікелей LLM провайдерінің API-на сұрау жіберуі (client-side қауіпсіз емес), немесе құпия кілттерді сақтау үшін Firebase Cloud Functions (`callFunction`) арқылы өтуі.
- Баптаулар `.env` файлында немесе Firebase Secret Manager ішінде сақталады.

## 2. Egov.kz және e-Otinish интеграциясы (Сілтемелік)
- Қазіргі уақытта `Templates.tsx` және кейбір құжаттар сілтемелік логикамен ҚР Электрондық үкіметіне немесе e-Otinish (Арыздар беру жүйесіне) бағыттайды. Ол үшін `<a href="..." target="_blank" rel="noopener noreferrer">` қолданылған. Бұл қауіпсіздік үшін (URL Phishing-тен қорғану) дұрыс амал.

## 3. Web Speech API (Жоспарда)
- Пайдаланушылар чатта мәтінді тек қолмен термей, дауыс арқылы енгізуі үшін Browser-built-in `SpeechRecognition` API-н қосу (Дауыстық чат) жоспарда бар. Ол Қазақ және Орыс тілдерін (Bilingual) тани алады (`lang="kk-KZ"`).

## 4. WhatsApp Business API (Заңгерлер бөлімінде)
- `Lawyers.tsx` бетінде адвокатқа тікелей жазу үшін WhatsApp-тың терең (deep link) сілтемелері қолданылады. Мысалы: `https://wa.me/7702...` Бұл ешқандай күрделі backend интеграцияны қажет етпестен, клиентті заңгермен лезде байланыстырады.

Бұл архитектура сыртқы сервистерге (мысалы, Kaspi төлемдері немесе 1C-бухгалтерия) тәуелділікті барынша азайтып, қосымшаны оңай орнатып-баптауды (plug-and-play) қамтамасыз етеді.
